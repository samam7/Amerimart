Amerimart Manager Pro V2

Upload index.html to your GitHub Pages repository.
If you already use products.js, keep products.js in the same folder as index.html.

New features included:
- Dashboard
- Price Checker with camera scanner
- Price change history and print label queue
- Cooler Planogram with doors/shelves
- Tap shelf to enlarge product details
- Scan product into shelf
- Backroom Pick List
- Inventory scan count
- Invoice photo/text parser and compare
- Daily Tasks with employee tracking
- Cigarette Order scan-outs
- Reports dashboard
- Database page with CSV import/export, backup, and Supabase settings

Important:
Use only Supabase anon/publishable key in this app. Do not paste a secret/service-role key.
